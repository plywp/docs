# PlyWP Documentation Site - User Guide

## Welcome to the New Professional Documentation Website!

This guide highlights all the features and improvements made to create a world-class documentation experience for the PlyWP project.

## 🎯 Site Structure

### Home / Landing Page
The landing page introduces PlyWP with:
- **Hero Section**: Eye-catching introduction with large typography and prominent call-to-action buttons
- **Features Section**: Showcase of core features (PlyWP Panel, Plyorde Daemon, Security, Developer Friendly)
- **Benefits Section**: 6 key reasons to choose PlyWP
- **Call-to-Action**: Direct link to installation guides
- **Documentation Preview**: Quick links to main documentation sections

### Documentation Hub (`/docs`)
A comprehensive resource organized into clear sections:

#### Installation Guides
- **Panel Installation**: Complete setup for the management panel
- **Plyorde Installation**: Daemon setup and configuration
- **Environment Guides**: Docker, Docker Compose, Kubernetes, and Manual deployment options

#### API Documentation
- **API Overview**: Understanding the API structure and capabilities
- **Authentication**: Security practices and token management
- **Endpoints Reference**: 15+ fully documented endpoints with examples
- **Code Examples**: Real-world examples in multiple languages

#### Guides & Best Practices
- **Architecture Guide**: Understanding PlyWP components and how they work together
- **Deployment Guide**: Best practices for production deployments
- **Security Guide**: Security considerations and hardening practices

## 🧭 Navigation Features

### Navigation Bar
Located at the top of every page:
- **Logo & Branding**: Click to return to home
- **Documentation Link**: Quick access to the docs hub
- **GitHub Link**: Direct link to the project repository
- **Search Icon**: Open global search dialog
- **Responsive Menu**: Mobile hamburger menu for smaller screens

### Sidebar Navigation (Docs Pages)
Appears on the left side of documentation pages:
- **Expandable Sections**: Collapse/expand major doc categories
- **Active Page Indicator**: Shows which page you're currently viewing
- **Breadcrumb Navigation**: Shows your location in the docs hierarchy
- **Quick Links**: Easy access to any documentation page

### Breadcrumb Trail
Shows your current location:
```
Docs / Installation / Panel
```
- Click any part to jump back
- Helps you understand where you are
- Last item is the current page
